# ebook downloader

一个国内可用的zlibrary下载工具

安装：
```
# npm i -g
```

使用：
```
Options:
  --source <string>     (default: "ylibrary")
  --type <type>         (default: "all")
  --page <number>       (default: "1")
  --ipfs <url>          (default: "https://gateway.pinata.cloud/ipfs")
  -s, --search <key>
  -d, --download <id>
  -o, --output <path>
  -h, --help           display help for command
```

配置：
```
位于 ~/.config/configstore/xgugugu-ebooks-cli.json
```
